# 📚 Библиотека преппера

**Библиотека знаний для автономного существования и восстановления после кризиса.**

> Запас еды даёт время. Библиотека знаний подскажет, что с этим временем делать.

---

## Зачем это нужно

Информации, которая может понадобиться в кризисной ситуации — огромное количество. Всю заранее не выучить и тем более не запомнить. Медицина, агрономия, строительство, электрика, консервация, ориентирование, связь — это тысячи страниц справочников.

Этот репозиторий — систематизированная коллекция **150+ книг, наставлений и справочников**, охватывающая все ключевые области знаний для выживания и восстановления цивилизации. Приоритет отдан русскоязычным источникам; англоязычные включены там, где русские не закрывают тему.

## Форматы файлов

Библиотека содержит файлы в следующих форматах:

- **PDF** — основной формат, сохраняет вёрстку и иллюстрации
- **DJVU** — сканы советских изданий (нужен просмотрщик, например WinDjView или Sumatra PDF)
- **EPUB / FB2** — электронные книги (удобно на планшетах и читалках)
- **DOC / DOCX** — отдельные методички и наставления

> 💡 Рекомендуется установить на ноутбук/планшет **Calibre** — универсальный менеджер электронных книг, который открывает все форматы и позволяет искать по содержимому.

## Как помочь проекту

- 🔍 **Нашли книгу, которой нет в списке?** Откройте Issue с названием, автором и кратким описанием
- 📂 **Есть качественный скан?** Предложите через Pull Request в соответствующий раздел
- 🌐 **Знаете перевод англоязычной книги?** Укажите в Issue — русскоязычные источники приоритетнее
- ⭐ **Поставьте звезду** — это помогает другим найти репозиторий
- ❓ **Противоречивая информация** — считаете что книга содержит недостоверную информацию или просто ифнормационный мусор, предложите Pull Request

## Лицензия

Репозиторий содержит только материалы, находящиеся в **общественном достоянии** (военные наставления, советские издания с истёкшим сроком охраны) или распространяемые авторами бесплатно. Коммерческие книги представлены только в виде **библиографических записей** (название, автор, описание) без файлов — их необходимо приобретать самостоятельно.

Если вы являетесь правообладателем и считаете, что ваши права нарушены — откройте Issue, и материал будет удалён.
